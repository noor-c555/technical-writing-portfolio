# Issue Tracking System Documentation

## Overview
The Issue Tracking System is a C++-based application designed to help users record, track, and manage software bugs throughout the development lifecycle. The system emphasizes clarity, usability, and reliability, providing structured workflows for logging, updating, and resolving issues.

## Documentation Included
- **Software Requirements Specification (Markdown)**: Defines system functionality, constraints, and requirements.  
- **Software Requirements Specification (PDF)**: Polished, formatted version of the SRS.  
- **User Manual (Markdown)**: Step-by-step instructions for end users.  
- **User Manual (PDF)**: Formal, polished user guide.
- **Testing Report (PDF)**: Official testing report, detailing test cases and outcomes.
